# Guia do Usuario

## Objetivo
Controlar o estoque com visao executiva no dashboard e operacao diaria de entradas, saidas e ajustes.

## Navegacao principal
- `Dashboard`: KPIs e grafico de fluxo diario.
- `Produtos`: cadastro e listagem.
- `Categorias`: cadastro e listagem.
- `Relatorios`: registro de movimentacoes e historico.
- `Configuracoes`: dados de sessao e notificacoes em tempo real.

## Fluxo recomendado
1. Entrar no sistema com credencial valida.
2. Cadastrar categorias e fornecedores (perfil Admin).
3. Cadastrar produtos.
4. Registrar movimentacoes no menu Relatorios.
5. Monitorar indicadores e alertas no Dashboard/Configuracoes.

## Boas praticas
- Definir `MinStock` coerente para cada SKU.
- Registrar `Reason` em toda movimentacao para auditoria.
- Usar perfil de funcionario para operacoes rotineiras.
